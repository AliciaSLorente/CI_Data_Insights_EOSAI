## MCP servers

Started automatically by `start_demo.py --mcp`.  
Can also be started manually for development:

```bash
python src/mcp_servers/submissions_server.py --transport sse --port 8601
python src/mcp_servers/scoring_server.py     --transport sse --port 8602
python src/mcp_servers/kg_server.py          --transport sse --port 8603
python src/mcp_servers/watcher_server.py     --transport sse --port 8604
```

Set `USE_MCP=true` in `.env` to route agent tool calls to the HTTP servers.  
With `USE_MCP=false` (default), the orchestrator uses inline Python functions.
